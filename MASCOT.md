# LumenJS Mascot: Lumi the Firefly

## Concept

**Lumi** is a friendly, glowing firefly — the official mascot of LumenJS. Fireflies are
nature's light-bearers, producing their own bioluminescence, making them the perfect
symbol for a framework named after light itself.

## Why a Firefly?

| Trait | Connection to LumenJS |
|-------|----------------------|
| **Bioluminescence** | Embodies "lumen" — Latin for light |
| **Guide in darkness** | LumenJS guides developers through full-stack complexity |
| **Small but powerful** | Lightweight framework with remarkable capability |
| **Flight patterns** | Echoes file-based routing and navigation |
| **Warmth** | Approachable, community-friendly open-source spirit |

## Visual Design

### Character

- **Body**: Stylized, slightly rounded firefly with a compact, friendly silhouette
- **Wings**: Two small, translucent wings with a subtle geometric pattern
- **Antennae**: Two thin, gently curved antennae adding personality
- **Eyes**: Large, expressive, and friendly
- **Glow**: Warm amber/golden light emanating from the abdomen — the signature element

### Color Palette

| Element | Color | Hex |
|---------|-------|-----|
| Body | Charcoal | `#2D2D3A` |
| Wing accent | Soft lavender | `#A8A4CE` |
| Glow (primary) | Warm amber | `#F5A623` |
| Glow (highlight) | Bright gold | `#FFD466` |
| Glow (outer) | Soft orange | `#F5A62340` (with transparency) |
| Eyes | White with charcoal pupils | `#FFFFFF` / `#2D2D3A` |

### Style Guidelines

- **Aesthetic**: Clean, geometric, modern — consistent with the Vite/Lit ecosystem
- **Minimum size**: 24x24px (favicon), scales up to any size
- **Simplified form**: At small sizes, reduce to a glowing dot with tiny wings
- **No harsh outlines**: Use soft edges and glow effects for warmth

## Poses & Usage

| Pose | Context | Description |
|------|---------|-------------|
| **Flying** | Dev server active, loading states | Lumi mid-flight, wings spread, trail of light |
| **Resting** | Idle, documentation headers | Lumi perched on a leaf or bracket `{` character |
| **Lighting a path** | Routing, navigation docs | Lumi flying ahead illuminating a dotted path |
| **Blinking** | Errors, warnings | Glow dims or shifts to a different color |
| **Swarm** | Community, ecosystem | Multiple small Lumis forming a constellation |

## Brand Voice

When Lumi "speaks" in CLI output, documentation, or error messages:

- Friendly and encouraging, never condescending
- Uses light/glow metaphors sparingly
- Example: `[LumenJS] Lumi is lighting up your dev server on port 3000`
- Error example: `[LumenJS] Lumi couldn't find that route — check your pages/ directory`
